# 项目概述 python1070

本项目是一个基于知识图谱的医疗问答系统，采用 Python 语言开发，整合了 Django 框架和 Vue.js 前端技术，实现了用户与管理员之间的互动问答功能。

## 技术栈

### 后端

- 开发语言：Python 3.7.7
- 框架：Django
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+
- 开发软件：PyCharm

### 前端

- Node.js
- Vue.js
- HTML

## 功能模块

### 系统角色

- 管理员
- 普通用户

### 管理员登录

管理员登录界面设计美观大方，易于操作。

### 后台首页

后台首页包含系统登录、用户、问答总数、年份等指标展示，菜单树和操作区域布局清晰。

### 医疗问答页面

医疗问答页面设计采用主流的人工智能客户服务界面，易于用户接受。

### 问答管理

支持搜索、编辑、删除功能，便于管理员管理和查询问答信息。

### 修改密码

提供当前最时髦的密码修改方法，确保用户信息安全。

### 个人信息

展示和管理个人信息，包括姓名、创建时间、最后修改日期等。

## 运行环境

- Python 3.7.7
- Django
- MySQL 5.7+
- Node.js
- Vue.js

## 部署步骤

1. 安装 Python 3.7.7 和 Django
2. 配置 MySQL 数据库
3. 安装 Node.js 和 Vue.js
4. 部署后端服务
5. 部署前端服务
6. 启动项目
